## Overview

Customer classification determines who can receive service and at what terms. All customers must align with the Charter and ROE (Return on Energy) principles.

---

## Class Definitions

| Class | Description | Approval | Notes |
|-------|-------------|----------|-------|
| **Alpha** | Early adopters, charter-aligned | Kenneth | First cohort |
| **Aligned** | ROE-positive, verified intent | Tiger validates | Standard service |
| **Neutral** | Unknown alignment | YELLOW gate | Requires review |
| **Misaligned** | Negative ROE or charter conflict | BLOCKED | No service |
